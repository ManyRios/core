# Environment Variables

## Core Runtime

- `VEIL_HOME`: override the base runtime directory
- `VEIL_PASSWORD`: bypass interactive password prompts
- `VEIL_REQUEST_TIMEOUT`: override request timeout handling

## Relay

- `VEIL_RELAY_PORT`
- `VEIL_RELAY_RATE_LIMIT`
- `VEIL_RELAY_REGION`

## Provider

- `VEIL_PROVIDER_HEALTH_PORT`
- `ANTHROPIC_API_KEY`
- `PROXY_URL`
- `PROXY_SECRET`

## Proxy

- `PROXY_PORT`
- `UPSTREAM_URL`

## Wallet and Crypto

- `VEIL_KDF_N`: KDF cost parameter override

## Logging and CLI Output

- `VEIL_LOG_LEVEL`
- `VEIL_LOG_FORMAT`
- `NO_COLOR`
- `FORCE_COLOR`

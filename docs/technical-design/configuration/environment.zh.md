# 环境变量

## 核心运行时

- `VEIL_HOME`：覆盖基础运行目录
- `VEIL_PASSWORD`：绕过交互式密码输入
- `VEIL_REQUEST_TIMEOUT`：覆盖请求超时处理

## Relay

- `VEIL_RELAY_PORT`
- `VEIL_RELAY_RATE_LIMIT`
- `VEIL_RELAY_REGION`

## Provider

- `VEIL_PROVIDER_HEALTH_PORT`
- `ANTHROPIC_API_KEY`
- `PROXY_URL`
- `PROXY_SECRET`

## Proxy

- `PROXY_PORT`
- `UPSTREAM_URL`

## Wallet 与密码学

- `VEIL_KDF_N`：KDF 成本参数覆盖

## 日志与 CLI 输出

- `VEIL_LOG_LEVEL`
- `VEIL_LOG_FORMAT`
- `NO_COLOR`
- `FORCE_COLOR`

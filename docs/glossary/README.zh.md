# 术语表

## 目的

本节定义 Veil 文档体系中稳定使用的公开术语。

## 核心术语

| 术语 | 统一写法 | 说明 |
|------|----------|------|
| gateway | 网关 | Consumer 使用的本地 OpenAI 兼容 HTTP 入口 |
| witness | 见证记录 | 用于用量证据、导出和后续对账的已签名记录 |
| discovery | 发现 | 查找和选择 Relay 节点的过程 |
| accountless access | 无账户接入 | 不让每个客户端直接绑定独立上游账户流程的使用方式 |
| privacy-preserving routing | 隐私保护式路由 | 通过角色拆分降低单点可见性，但不承诺完美匿名 |
| capacity | 容量 | Provider 暴露出来的可执行供给能力 |
| quota | 配额 | 上游账户额度或 Provider 侧可用上限 |
| settlement | 结算 | 把请求与见证记录证据转换成可计费或可支付结果 |
| build loop | 开源构建循环 | 围绕 Veil 开发形成的开源贡献与评审循环 |

## 写作规则

- 在文档里优先使用统一写法
- 遇到容易歧义的词，在局部重新定义其语境
- 不要用会改变信任边界或产品含义的营销化近义词
